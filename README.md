# 🎟️ EventSphere – Smart Event Management & Ticket Booking System

EventSphere is a full-stack web application developed using ReactJS, Node.js, Express, and SQLite.  
The platform helps organizers create and manage events while allowing attendees to register, book tickets, and track bookings in real time.

---

# 🚀 Live Features

✅ User Authentication (Register/Login)  
✅ Role-Based Access (Attendee / Organizer / Admin)  
✅ Event Creation & Management  
✅ Event Search & Filtering  
✅ Ticket Booking System  
✅ Event Capacity Tracking  
✅ Booking History  
✅ QR-Based Ticket Generation  
✅ Responsive UI Design  
✅ Protected Routes using JWT Authentication  
✅ SQLite Database Integration
